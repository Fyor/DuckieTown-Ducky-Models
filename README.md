- [static32](https://universe.roboflow.com/duckies-in-simulations/unity-duckies-static-lighting)
- [dynamic32](https://universe.roboflow.com/duckies-in-simulations/unity-duckies-dynamic-lighting)
- [extreme32](https://universe.roboflow.com/duckies-in-simulations/unity-duckies-extreme-dynamic-lighting)
